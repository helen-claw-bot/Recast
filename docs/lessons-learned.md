# AI 生图 & 生视频 踩坑复盘

> 基于《回笼》天台试探 + 宋祈年重逢两场实战，2026-03-24 ~ 2026-03-25

---

## 一、生视频（可灵 Kling）

### 1. 动作、口型、台词对不上
- 人在喝水，台词已经出来了——动作和语音的时间轴不同步
- **Workaround**: 拆分为无台词动作段 + 纯台词段，分别生成后剪辑拼接

### 2. 人物语气、重音、动作不稳/不充分
- 首次生成的表情更浮夸，语气更轻浮——AI 默认"演过头"
- 台词重音不在情感点上，也不准确——可灵 TTS 不理解情绪重音
- 本属于 A 的台词，生成结果是 B 来讲——角色台词串台
- **Workaround**: 用标点手动控制节奏：逗号=微停顿，省略号=拉长停顿，把重音词孤立出来
  - 原：`"那我还有一件事也是故意的。"`
  - 改：`"那我，还有一件事……也是，故意的。"`

### 3. 无台词片段中，AI 自行加入无意义人声或台词
- 明确标注"无台词"也不一定管用
- **Workaround**: 后期剪辑时替换音轨（静音 + 环境音覆盖）

### 4. Prompt 驱动失效
- **约束过多时产生 conflict**：AI 随机忽略一些动作
- **视频时长不够**：AI 没时间执行所有动作，随机跳过
- **无明确原因**：AI 随机忽略某些指令
- **Workaround**: 减少单段 prompt 的动作数量（3-4 个动作/段为上限），给足时长

### 5. AI 自由发挥
- 导致：身体僵硬、目光涣散/呆滞、身体朝向不对、视角不对
- AI 在没有明确指令的地方会自行补充，且补充质量不稳定

### 6. 正反打特写时背景不一致
- 首尾帧分别为不同人物特写时，AI 会为每个角色随机生成背景
- **长镜头反而更好**：1B 电梯→走廊→推门的一镜到底效果远好于短镜头拼接，因为 AI 不需要跨镜头保持背景一致性
- **Workaround**: 尽量用长镜头/跟拍，避免频繁正反打切换

## 二、生图（Gemini）

### 1. 多轮对话后图片退化
- Gemini 在多轮对话后，生成的图片会逐渐变形、模糊
- **Workaround**: 定期开新对话，把确认好的 prompt 粘贴到新 session 重新生成

### 2. 环境参考图的质量决定一切
- 最初的环境参考图必须高清，否则直接拉低后续所有生成图的清晰度
- 源头质量 = 天花板

### 3. 人物一致性不稳定
- 面部在不同构图/角度下会"变脸"
- **Workaround**: 在 prompt 最开始声明 `Photo N: Character A (Name)`，可解决约 70% 的一致性问题
- 剩下 30% 只能靠多次重试 + 挑选最接近的

### 4. Prompt conflict 导致物理错误
- 出现物理现象错误：人靠在空气上、平白无故多一面墙、人大于建筑比例
- **AI 会严格执行每个词**：`surrounds` → 画四面围墙，`almond nails` → 画美甲级长指甲
- **AI 不会推理未说的内容**：不说手臂方向就默认水平，不说酒的颜色就随机
- **Workaround**: 
  - 空间词慎用（`surrounds`, `around`, `encircle` 会被严格执行）
  - 排除项和正面描述同等重要（`no veins, no knuckle bones, no dirt`）
  - 每个镜头都要重复描述道具颜色/细节，不能假设 AI 会记住

### 5. 手部生成顽疾
- 默认生成偏男性化的手（宽掌、粗指节）
- `almond-shaped nails` → 美甲级别长指甲，不是短自然甲
- **Workaround**:
  - 必须加性别+族裔修饰：`young East Asian woman's hand`
  - 排除项：`no visible veins or knuckle bones`
  - 指甲用：`short trimmed natural nails, clean pink nail beds`（不要用 almond）
  - 手臂角度必须明确入画方向：`forearm enters frame from upper right, nearly vertical`

### 6. 色彩/光线问题
- 暖色灯光 → 牙齿跟着染黄，AI 不会单独处理牙齿白度
- 梅酒颜色在不同帧间不一致
- **Workaround**: 生成阶段无法精确控制，后期用 HSL 二级调色统一处理

## 三、通用经验法则

| 法则 | 说明 |
|------|------|
| **AI 忠实执行你说的每个词** | "surrounds" 就真围一圈，"almond nails" 就真做美甲 |
| **AI 不会推理你没说的** | 不说手臂方向就默认水平，不说酒颜色就随机 |
| **排除项 = 正面描述** | "no veins, no dirt" 比只说 "feminine" 有效 |
| **参考图 > 文字 prompt** | 实物参考图直接解决文字描述不清的问题 |
| **长镜头 > 短镜头拼接** | 避免跨镜头一致性问题，让 AI 在一段内自洽 |
| **单段动作 ≤ 3-4 个** | 超过就会 conflict，AI 随机丢弃 |
| **每段重复道具描述** | AI 不记住上一段的设定 |
| **新 session 保鲜** | 多轮对话后质量退化，定期换新 |
| **后期能修的不纠结** | 牙齿颜色、微调色温等，后期30秒搞定 |

## 四、推荐工作流

```
1. 高清环境参考图 → 确定基础画面质量
2. 角色定妆照 → Photo N: Character A 锁定人物
3. 分镜 prompt → 单段 ≤ 3-4 个动作，明确入画方向/空间关系
4. 生图验证 → 新 session 生成，多轮后换新
5. 首尾帧确认 → 可灵图生视频
6. 长镜头优先 → 减少正反打，一镜到底
7. 台词标点断句 → 手动控制重音和节奏
8. 后期统一处理 → 调色、音频替换、穿帮修复
```
